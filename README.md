# NeoConvert
NeoConvert – Conversor de Moedas Online Um projeto moderno e intuitivo desenvolvido com HTML, CSS e Python (Flask) para realizar conversões de moedas em tempo real.

▶️ Como Executar o Projeto
1. Criar um ambiente virtual (opcional, recomendado)
Linux/macOS:
python3 -m venv venv
source venv/bin/activate
Windows:
python -m venv venv
venv\Scripts\activate
2. Instalar as dependências
pip install -r requirements.txt
3. Executar a aplicação
python app.py
